### SQL SESSION SIMPLE / BATCH 모드

**일반 모드 vs 배치 모드의 차이:**

| 구분          | 일반 모드 (SIMPLE)           | 배치 모드 (BATCH)                               |
| ------------- | ---------------------------- | ----------------------------------------------- |
| **실행 시점** | Mapper 메서드 호출 즉시 실행 | `flushStatements()` 호출 시 일괄           실행 |
| **DB 통신**   | 매번 개별 통신               | 한 번에 모아서 통신                             |
| **성능**      | 느림 (건수가 많을 때)        | 빠름 (네트워크 오버헤드 감소)                   |
| **반환값**    | 영향받은 행 수 반환          | 0 반환 (즉시 알 수 없음)                        |
| **사용 예시** | 일반적인 CRUD                | 대량 insert/update                              |





### SqlSession / SqlSessionTemplate

- SqlSession
  - 인터페이스
- SqlSessionTemplate
  - SqlSession을 구현
  - Spring-Mybatis를 사용하면 기본적으로 SqlSessionTemplate을 사용함
