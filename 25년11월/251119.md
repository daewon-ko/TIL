# 251119



### Index

- '**추가적인 쓰기작업과 저장 공간을 활용**하여' 데이터베이스의 테이블 검색 속도를 향상시키기 위한 '**자료구조**'



- Index가 걸려있는 컬럼이 있을때, Insert 쿼리가 수행된다면?
  - PK를 제외한 모든 컬럼의 Index는 'Secondary Index'이다.
  - Secondary Index의 리프노트엔 다음의 2가지가 존재
    - Index Key
      - 해당 컬럼에 삽입된 새로운 값
    - Pointer
      - 실제 데이터 레코드가 저장된 위치를 가리키는 포인터
