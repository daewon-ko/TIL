### Shell과 Exit Code 

Intelij 등의 IDE를 사용하도 로컬 PC내에 설정한 별도의 Shell을 사용가능

맥에서는 통상 Zsh, 윈도우에서는 Powershell을 사용

- Shell환경
  - 프로세스가 특정 신호(Signal)로 인해 종료되면, 해당 신호 번호에 128을 더하여 종료 코드로 반환한다.
  - 해당 신호 번호에 128을 더한 값을 종료코드로 반환하는 관례가 있음
- SIGINT(Interupt Signal)
  - 2
    - 'Ctrl+C'를 누른 것과 동일
    - 즉, 외부중단을 의미
  - ExitCode : 130
- SIGTERM
  - 15
  - Graceful Shutdown
    - 커널이 Process에게 일정시간을 주고 종료되기까지 기다리다, 종료가 안되면 SIGKILL을 보내서 강제종료 
  - ExitCode : 143
- SIGKILL
  - 9 
  - 강제 종료
  - ExitCode : 137
- Exit Code 1의 의미
  - 개발자가 명시적으로 프로그램 내에서 System.exit(1)을 호출하는 경우
  - main 메서드 실행 중 JVM이 비정상적으로 종료되었을때
    - NPE와 같은 Uncaught Exception발생시
