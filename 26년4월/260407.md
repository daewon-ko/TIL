### Nginx / ALB / LB 등에서 API서버 호출 시 Header 이슈



- X-Forwared-For 

  - (N개의) 실제 클라이언트 IP
  - 프록시를 여러번 거치면 IP 체인을 여러개 구성 가능(프록시 체인)

-  X-Real-IP

  - (단일한) 실제 클라이언트 IP

- 둘 모두 Nginx가 실제 클라이언트 IP를 HTTP Header에 담을때 사용하는 Header정보

  

클라에서 API서버로 직접 호출 할경우, 사용자가 임의로 헤더를 조작해서 X-Forwared-For직접 제어가능한가?

그렇다면 이거에 대해서 방지는 어떻게 해줄수 있는가?
