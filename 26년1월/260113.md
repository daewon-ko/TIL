### TS ORM 사용시 null과 Undefined차이

```
      const xxx = await this.prisma.xxx.update({
        where: { xxx_uid: aly_pet_uid },
        data: {
          some_field: some_field ? encryptAES(some_field) : undefined, 
        },
      });
      
      const xxx = await this.prisma.xxx.update({
        where: { xxx_uid: aly_pet_uid },
        data: {
          some_field: some_field ? encryptAES(some_field) : null, 
        },
      });
```



- undefined로 3항 연산자 유지시 update쿼리 수행시에 값이 없으면 해당 필드는 무시됨
- 그러나 null로 구성할 경우에 해당 필드 자체를 null로 엎어침
