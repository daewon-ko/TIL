# 260224



### Explain Type 'ref'

- ref        → 비유니크 인덱스 동등 조건 탐색

  - ex) orders Table에 user_id 컬럼이 있고 해당 컬럼에 인덱스가 있다고 하면, 
    select * from orders where user_id = 'test123';라는 쿼리가 있다고 하자. 
    user_id에 인덱스를 걸어놓으면 user_id에 대해서 동등조건으로 인덱스가 탐색할 것이다. 
    그러나 해당 유저가 주문을 다건 했을 수도 있으므로 유니크하진 않다. 
    따라서 '비유니크 인덱스 동등' 조건탐색이다.
